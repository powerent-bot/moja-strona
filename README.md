# Prosta strona na GitHub Pages (za darmo)

To repo zawiera gotową stronę (HTML/CSS) z podstronami: **Regulamin**, **FAQ**, **Polityka prywatności**, **Kontakt**.

## Jak opublikować na GitHub Pages

### Wariant A — domena `twojlogin.github.io`
1. Wejdź na GitHub i utwórz **publiczne** repo o nazwie: `twojlogin.github.io` (zastąp `twojlogin` swoim loginem).
2. Wgraj wszystkie pliki z tego folderu (index.html, itd.).
3. Wejdź w **Settings → Pages** i ustaw **Branch: `main` / folder `/ (root)`**.
4. Po chwili strona będzie dostępna pod: `https://twojlogin.github.io/`.

### Wariant B — podadres `twojlogin.github.io/nazwa-projektu`
1. Utwórz **publiczne** repo `nazwa-projektu`.
2. Wgraj pliki.
3. **Settings → Pages → Branch: `main`**.
4. Strona będzie pod `https://twojlogin.github.io/nazwa-projektu/`.

## Formularz kontaktowy (Formspree — darmowy)
1. Wejdź na https://formspree.io i utwórz formularz (Free plan).
2. Skopiuj adres endpointu (wygląda jak `https://formspree.io/f/abcdwxyz`).
3. Otwórz `kontakt.html` i podmień `action="https://formspree.io/f/XXXXYYYY"` na Twój adres.
4. Zapisz zmianę w repo — formularz zacznie działać.

## Co jeszcze zmienić
- W plikach podmień **Twoja Marka**, adres e-mail i telefon.
- Zaktualizuj treści **Polityki prywatności** i **Regulaminu** pod swój przypadek.
- Jeśli **nie używasz analityki/cookies** — nic nie musisz dodawać.
- Jeśli dodasz analitykę — zaktualizuj sekcję cookies i rozważ baner zgody.
